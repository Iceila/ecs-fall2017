HTML Reference

This entire website is coded in HTML. But if you don't know HTML, you can't figure out how this website is made. For my own reference and yours, these are the basics of HTML.


Boilerplate Code

The code below is the standard code needed to make anything related to HTML.

  
    <!DOCTYPE html>
    <html>

    <head>
      <title>This is a Title</title>
    </head>

    <body>

      </h1>This is a heading</h1>
      <p>This is a paragraph.</p>

    </body>

    </html>

<!DOCTYPE html> tells your HTML file that you'll be using HTML on this document.
<html> is the place you'll put all your HTML code in.
<head> is where all the elements that don't show up direcly on the page go.
<title> is one example of such an element. This page's title is HTML Reference.
<body> is where you find all the code that makes up the actual webpage.
