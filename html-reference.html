HTML Reference

This entire website is coded in HTML. But if you don't know HTML, you can't figure out how this website is made. For my own reference and yours, these are the basics of HTML.

Boilerplate Code

The code below is the standard code needed to make anything related to HTML. It should be the first thing you type or paste into your HTML file.

  
    <!DOCTYPE html>
    <html>

    <head>
      <title>This is a Title</title>
    </head>

    <body>

      </h1>This is a heading</h1>
      <p>This is a paragraph.</p>

    </body>

    </html>

<!DOCTYPE html> tells your HTML file that you'll be using HTML on this document.

<html></html> is the place you'll put all your HTML code in.

<head></head> is where all the elements that don't show up direcly on the page go.

<title></title> is one example of such an element. This page's title is HTML Reference.

<body></body> is where you find all the code that makes up the actual webpage.

<body> Commands

All these commands allow you to add information to your site.

Headings <h1></h1>

The <h1> automatically format your text on your web page to be more noticable than a paragraph, by increasing the font size and bolding it. The text on the top of the page is an example of <h1>. There are six heading styles, each one progressively smaller than the last heading.


<h1>This is a heading</h1>

Paragraphs <p></p>

This is a paragraph. Everything that's got normal text is a normal paragraph. Paragraphs have no font style, unless you bold the text on your own. Other ways you can style a paragraph is by italicizing or underlining.

<p>This is a paragraph!</p>

This is a paragraph!


<br> is another way to edit paragraph styles. It indents a line from the previous block of information.

Comments <!-- -->

When you're coding, you'll often find you need to remind yourself of something, whether it's to fix a certain issue or to give a coworker clarificaiton. Comments are lines of code that the rendering software ignores, and is only visible in your source code. HTML codes are specified as so:

<!-- This is a comment! -->

Links <a href=""></a>

This tag is also known as the anchor element. Adding the href attribute makes it linkable. You can link text, images, or just about anything you want in HTML. Navigation bars at the top of a webpage are made with links. This is the syntax of the anchor tag:

<a href="index.html">This is a link to my homepage</a>

You put the URL or local link in between the quotations. This is a link to my homepage.

Images <img src="">

To embed images into the page using HTML, we use the <img> tag. If you want some visuals to break up the text on your website, images can help you do that. The basics you need for an image tag is the tag itself and the src or source attribute. The source can be from inside your website or a URL from another website.

<img src="/images/Russet_potato_cultivar_with_sprouts.jpg" alt="Russet Potatoes" width="437.8" height="221.1">

Other attributes that you can add to the img tag are the alt attribute, which makes text show up if the image does not load. In this case, it is "Russet Potatoes". The width and height attributes allow you to resize the image on your website.

Lists <ol><, <ul></ul>, <li></li>

HTML also offers the feature of lists. It automatically formats whatever you put so it looks like a list. If you make a navigation bar, you need a list. If you're just putting a bunch of links on your webpage, lists are good for organizing those links. There are two types of lists. <ol> and <ul>. Ordered lists have numbers as their default list type. Unordered lists have a black circle, or bullet as their default list type. The <li> tag separates each list item.

This is an ordered list:

  
  <ol>
    <li>This is list item number one.</li>
    <li>This is list item number two.</li>
    <li>This is list item number three.</li>
  </ol>
  1. This is list item number one.
  2. This is list item number two.
  3. This is list item number three.

This is an unordered list:

  
  <ul>
    <li>This is list item number one.</li>
    <li>This is list item number two.</li>
    <li>This is list item number three.</li>
  </ul>
  • This is list item number one.
  • This is list item number two.
  • This is list item number three.

Tables <table></table>, <tr></tr>, <th></th, <td></td>

Another way to represent information is by table. The four commands above are needed to create a table. Tables are basically lists in a more structured format. You can have multiple columns of information. The code for a table is nested in the <table> element. <tr> makes a new table row. Table headers are defined by the <th> tag. Finally, the <td> allows you to make a description for your table. All the elements together to make a table look something like this:

  
  <table>

    <tr>
      <th>Heading 1</th>
      <th>Heading 2</th>
      <th>Heading 3</th>
    </tr>
    <tr>
      <td>Description 1</td>
      <td>Description 2</td>
      <td>Description 3</td>
    </tr>

  </table>
Heading 1 Heading 2 Heading 3
Description 1 Description 2 Description 3
