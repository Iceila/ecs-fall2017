Attributes

Most HTML tags can have some type of attribute. For example, the anchor tag <a> can have the attribute 'href', which allows a link to be inserted into the anchor tag. Another example is the images tag, or <img>, which has the attribute 'src' to link the source image, 'alt' to show alternate text, and the 'width' and 'height' attributes. Some tags can work without attributes, others absolutely need their attributes.
